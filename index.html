Notion 데이터베이스
업로드 매뉴얼

WYD DID 웹사이트 콘텐츠 관리 가이드

공통 주의사항

  • 데이터 반영 시간: 업로드 후 최대 60초 후에 웹사이트에 반영됩니다(3월 31일 이후 1시간으로 늘릴 예정)
  • 한글/영문 별도 관리: 각 언어별로 별도의 데이터베이스가 있습니다
  • 빈 값 주의: 필수 속성이 비어있으면 해당 항목이 표시되지 않을 수 있습니다
아래로 스크롤하세요
01

캘린더 (Calendar)

속성 구조

속성명 타입 필수 설명
제목 Title ✅ 이벤트 제목
날짜 Date ✅ 이벤트 날짜
소제목 Rich Text ❌ 부가 설명

작성 방법

  1. 제목: 이벤트명 입력 (예: "청년 미사")
  2. 날짜: 단일 날짜 또는 "End date" 활성화하여 기간 설정
  3. 소제목: 추가 설명이 필요한 경우 입력

주의사항

  • 날짜가 없으면 목록에 표시되지 않습니다
  • 최신 날짜순으로 자동 정렬됩니다
02

공지사항 (Notice)

속성 구조

속성명 타입 필수 설명
타이틀 Title ✅ 공지사항 제목
날짜 Date ✅ 작성 날짜
태그 Rich Text ❌ 카테고리 (공지, 소식)
파일과 미디어 Files ❌ 썸네일 이미지

작성 방법

  1. 타이틀: 공지사항 제목 입력
  2. 날짜: 작성일 선택
  3. 태그: 카테고리 입력 (공지, 소식, 안내 등)
  4. 파일과 미디어: 썸네일 이미지 업로드 (이미지는 클수록 좋으며, 비율을 맞춰서 업로드)
  5. 상세 페이지: 타이틀 옆 "열기" 버튼 클릭하여 본문 작성(노션에서 작성 후 붙여넣기 권유)
03

유튜브 (YouTube)

속성 구조

속성명 타입 필수 설명
제목 Title ✅ 영상 제목
날짜 Date ✅ 업로드 날짜
유튜브 동영상 고유 id Rich Text ✅ YouTube Video ID

YouTube Video ID 찾는 방법

전체 URL:

https://www.youtube.com/watch?v=dQw4w9WgXcQ

짧은 URL:

https://youtu.be/dQw4w9WgXcQ

→ Video ID: dQw4w9WgXcQ (이것만 입력!)

주의사항

  • 전체 URL을 입력하면 안 됩니다! Video ID만 입력
  • Video ID는 보통 11자리 영문+숫자 조합
  • 비공개/삭제된 영상은 재생되지 않습니다

YouTube 미리보기

Video ID를 입력하면 미리보기가 표시됩니다
04

FAQ

속성 구조

속성명 타입 필수 설명
아이디(아무거나 쓰세요) Title ✅ 고유 ID
제목 Rich Text ✅ 질문
내용 Rich Text ✅ 답변

작성 방법

  1. 아이디: 순번이나 고유값 입력 (예: "1", "faq-001")
  2. 제목: 질문 내용 입력 (예: "참가비는 얼마인가요?")
  3. 내용: 답변 내용 입력

주의사항

  • 아이디는 중복 가능하지만, 구분을 위해 고유하게 작성 권장
  • 긴 답변은 줄바꿈(Shift + Enter)으로 가독성 높이기
  • 데이터베이스 순서대로 표시 (드래그로 순서 변경 가능)
05

관련 사이트 (Site)

속성 구조

속성명 타입 필수 설명
사이트명 Title ✅ 사이트 이름
URL URL ✅ 사이트 링크 주소
파일과 미디어 Files ✅ 사이트 로고/썸네일

작성 방법

  1. 사이트명: 사이트 이름 입력 (예: "WYD 서울 2027")
  2. URL: 전체 주소 입력 (예: https://wydseoul2027.kr)
  3. 파일과 미디어: 로고 또는 대표 이미지 업로드 (이미지는 클수록 좋으며, 비율을 맞춰서 업로드)

주의사항

  • URL은 https:// 포함 전체 주소 입력
  • 이미지가 없으면 깨진 이미지로 표시될 수 있음
06

이미지 업로드 팁

방법 1: 직접 업로드

  1. 파일과 미디어 속성 클릭
  2. "Upload" 선택
  3. 컴퓨터에서 이미지 선택

방법 2: 외부 링크 사용

  1. 파일과 미디어 속성 클릭
  2. "Link" 선택
  3. 이미지 URL 붙여넣기

권장 이미지 사양

이미지는 크면 클수록 좋습니다! 각 용도에 맞는 비율을 반드시 맞춰서 업로드해주세요.

용도 최소 크기 비율 최대 용량
공지사항 썸네일 1920x1080px 이상 16:9 5MB
관련 사이트 로고 1000x1000px 이상 1:1 2MB
07

문제 해결

Q 업로드했는데 웹사이트에 안 보여요
  • 60초 기다리기: 캐시 갱신 시간(3월 31일 이후 1시간으로 변경)
  • 필수 속성 확인: 빈 값이 없는지 확인
  • 새로고침: 브라우저 캐시 문제 (Ctrl/Cmd + Shift + R)
Q 이미지가 안 보여요
  • Notion 내부 이미지 URL은 1시간 후 만료됨
  • 외부 이미지 링크 사용 권장 (R2, S3 등)
  • 이미지 파일 형식 확인 (PNG, JPG, WebP 지원)
Q 영문 페이지에 안 보여요
  • 영문 데이터베이스(_EN)에도 동일하게 입력했는지 확인
Q 순서를 바꾸고 싶어요
  • 캘린더/공지/유튜브: 날짜순 자동 정렬 (변경 불가)
  • FAQ/사이트: 데이터베이스에서 드래그하여 순서 변경

문의사항이 있으시면 담당자에게 연락해주세요.
